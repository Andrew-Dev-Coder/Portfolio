Developing Summary

Initial Design

This webpage was initially developed for my transition task for BTEC IT from Year 11 to Year 12. This was also my first time developing a project for a client. The first version of the webpage was not accessible or responsive.

Refactoring

I decided to refractor the website so that it was responsive and accessible.

First Version

Planning

I started by brainstorming in a digital notebook about what features would be needed which was shared with the client. Then I came up with a colorless outline. Once the client was happy with the colorless outline, I then added colour to the outline. I did not make a wireframe because that this point I was unaware of them.

Developing

When I made the first version of the website I had very little knowledge of HTML and CSS, and so did the FutureLearn: A Beginner’s Guide to Web Development with HTML5 alongside. Looking back, I should have completed the course before beginning to write the HTML.
I have picked up CSS from setting a goal to achieve, such as responsive design and then researching on how to implement that feature. I mainly used w3schools to achieve this.
