Development Summary

Initial Design

This webpage was initially developed for my transition task for BTEC IT from Year 11 to Year 12. This was also my first time developing a project for a client. The first version of the webpage was not accessible or responsive.

Refactor

I decided to refractor the website so that it was responsive and accessible.

First Version

Planning

I started by brainstorming in a digital notebook about what features would be needed which was shared with the client.
Then I made a colorless outline. Once the client was happy with the colorless outline, I then added colour to the outline. I did not make a wireframe because at this point I was unaware of them.

Developing

When I made the first version of the website I had very little knowledge of HTML and CSS, and so did the FutureLearn: A Beginner’s Guide to Web Development with HTML5 alongside. Looking back, I should have completed the course before beginning to write the HTML.
I have picked up CSS from setting a goal to achieve, such as responsive design and then researching on how to implement that feature. I mainly use w3schools to achieve this.

Second Version

Planning

I started by looking at the previous website and the original outlines I created. Then I decided to create issues on GitHub to allow for me to do light planning for what needed changing and adding. I also made a GitHub project which I linked the issues to.

Developing

I started by removing all of the previous HTML as it the code was such a mess that it was easier to start over. I then made the head section, the biggest change was adding Open Graph. Once I had completed the head section, I added the logo and brand name including a custom font from Google Fonts.
