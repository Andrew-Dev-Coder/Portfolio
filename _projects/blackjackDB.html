Planning

I started by copying my original blackjack game code into a new file. I then looked into different database formats and looked for the simplest one to use with Python because this was my first time using a database with Python. Once I had found a database engine that worked with Python in a simple manner I practiced connecting to a python program, I did this by creating a practice database and running SQL commands from a python file.

Developing

To learn how to use the Python sqlite3 module I create a small mock database and used a Python file and practiced using it to execute SQL. I did the same process of finding a module, practicing in a separate python file when I looked for a password hashing module, for this program I settled on Agron2.
