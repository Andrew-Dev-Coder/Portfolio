Testing

I tested whenever I made new functions, I tested them individually and with the rest of the code at that point. If there was an issue which caused the code to have a logic or syntax error I would then look through the code to see if I could spot the issue, but if I was unable to do this I would then use debugging with breakpoints to identify where the issue came from. Once I believed I had completed the code, I did final tests and then got some of friends to also test the code (beta testing).
